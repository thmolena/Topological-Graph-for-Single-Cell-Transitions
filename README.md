# Topological Graph Learning for Single-Cell State Transitions

Label propagation on a kNN cell graph with topology-aware active sampling of rare states under leakage-controlled splits.

[Live in-browser demo](https://thmolena.github.io/P4-Topological-Graph-RL-for-Single-Cell-Transitions/) · Python 3.9–3.13 · MIT License

## Abstract

A single-cell dataset admits a graph representation in which each cell is joined to its
nearest neighbours on the expression manifold, and cell-state transitions trace paths along
that graph. This project forecasts the state of held-out cells by propagating a small set of
annotated labels across a symmetric k-nearest-neighbour cell graph (graph-smoothed label
propagation), and complements forecasting with a topology-aware active-sampling policy that
selects which unlabeled cells to annotate next in order to recover a deliberately rare cell
state. The benchmark is a seeded synthetic branching lineage carrying donor, batch,
pseudotime, and perturbation metadata; transfer is evaluated under four leakage-controlled
held-out split protocols, each verified to keep the held-out group out of the annotated set.
The number of connected components of the cell graph (a Betti-0 / `H_0` proxy) is asserted to
be monotone non-increasing in `k`. Every reported figure is generated by the included
pipeline and traced by an audit gate; none is hand-edited.

## Principal contributions

1. **Graph-smoothed forecasting under label scarcity.** Label propagation on the kNN cell
   graph forecasts held-out cell states from a small annotated set and is compared, on an
   identical annotated set and budget, against a non-graph k-nearest-neighbour classifier.
2. **Topology-aware active sampling for rare states.** A policy scoring cells by uncertainty
   weighted by inverse local density selects annotation targets and is compared against random
   querying for rare-state recall at a fixed labelling budget.
3. **Leakage-controlled evaluation.** Four held-out split protocols (donor, batch, pseudotime,
   perturbation condition) are accompanied by an explicit leakage check, recorded as an
   integrity flag in the results artifact.
4. **Topological invariant as a tested property.** The Betti-0 curve of the cell graph is
   asserted to be monotone non-increasing in `k` on seeded data, with the assertion encoded as
   a test.
5. **Auditable reproducibility.** A readiness gate verifies that each reported headline value
   traces to a generated artifact, that integrity flags hold, and that forbidden
   absolute-superiority phrasing is absent.

## Headline results

The values below are transcribed verbatim from `code/results/summary.json` and
`code/results/main_results.md`. They correspond to the **reported scale**
(`code/configs/full.yaml`): 3000 cells, 4 states, rare-state fraction 0.04, `k = 20`,
label fraction 0.03, 5 seeds; mean wall-clock runtime 20.802 s and peak memory 320.64 MB on
the recorded platform.

| split | method | accuracy | ECE | rare-state recall |
|---|---|---|---|---|
| donor | **graph-smoothed** | **0.7004** | 0.1247 | **0.1520** |
| donor | baseline kNN | 0.6550 | 0.1073 | 0.0000 |
| batch | **graph-smoothed** | **0.6995** | 0.1383 | **0.4618** |
| batch | baseline kNN | 0.6355 | 0.0905 | 0.0320 |
| time | **graph-smoothed** | **0.5474** | 0.1328 | **0.0000** |
| time | baseline kNN | 0.4574 | 0.0970 | 0.0000 |
| perturbation | **graph-smoothed** | **0.6875** | 0.1649 | **0.3393** |
| perturbation | baseline kNN | 0.6490 | 0.1184 | 0.0286 |

Aggregated across the four split protocols (from the `headline` block of `summary.json`):

- Graph-smoothed mean accuracy **0.65868** against baseline mean accuracy **0.599222**, a gain
  of **0.059458**.
- Graph-smoothed mean rare-state recall **0.238284** against baseline mean rare-state recall
  **0.015143**.
- Topology-aware active sampling reaches rare-state recall **0.355579** against **0.024673**
  for random querying at the same labelling budget, a gain of **0.330906**.

Graph propagation exchanges a measure of calibration (higher ECE on several splits) for the
accuracy and rare-recall gains; both directions are reported in the table above. The Betti-0
curve, per-split confidence intervals, and integrity flags are written to
`code/results/summary.json` and rendered in `code/figures/`.

A laptop-scale **smoke** configuration (`code/configs/smoke.yaml`: 400 cells, single seed) is
provided for fast checks; its numbers are demo-scale and differ from the reported scale above.

## Installation

```bash
pip install topocell
```

From source, for development:

```bash
cd code
pip install -e .            # CPU reference implementation
pip install -e .[gpu]       # optional accelerated backend (torch)
```

## Reproduction

The installed console entry point regenerates the results artifact, tables, and figures from a
deterministic seeded run:

```bash
topocell-reproduce                              # reported scale (configs/full.yaml)
topocell-reproduce --config configs/smoke.yaml  # laptop-scale smoke run
topocell-reproduce --skip-run                   # tables and figures from existing summary.json
```

Run the entry point from the `code/` directory, which contains `scripts/`, `configs/`, and
`results/`. The Makefile targets in `code/` expose the same pipeline step by step
(`make demo`, `make tables`, `make figures`, `make audit`, `make full-run`).

### Library modules — `code/src/topocell/`

| Module | Role |
|---|---|
| `synthetic.py` | Seeded synthetic branching lineage: cells along a tree of fates over pseudotime, carrying branch/state, donor, batch, perturbation, and a rare state. |
| `ingest.py` | Stub hooks for AnnData / perturbation-screen ingestion (scanpy), defining where measured data attaches. |
| `graph.py` | Symmetric kNN cell graph, its component count (Betti-0 / `H_0` proxy) over `k`, and local density. |
| `splits.py` | The four held-out split protocols (donor, batch, time, perturbation) with an explicit leakage check. |
| `policy.py` | Graph-smoothed label propagation, the kNN baseline, and the topology-aware active sampler against random querying. |
| `metrics.py` | Accuracy, expected calibration error, rare-state recall, and per-split summaries. |
| `runner.py` | End-to-end protocol driver; writes `results/summary.json`. |
| `audit.py` | Readiness gate: traceable-number and forbidden-claim checks. |
| `plotting.py` | Lineage scatter and forecast / rare-recall figures. |
| `reproduce.py` | Installed reproduction command backing the `topocell-reproduce` entry point. |
| `config.py`, `seed.py` | YAML configuration loading and deterministic seeding with run provenance. |
