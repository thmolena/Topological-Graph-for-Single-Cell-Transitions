Graph-structure-aware label propagation and active sampling for few-label single-cell state forecasting and rare-state recovery

A controlled, fully seeded benchmark isolates a single variable — whether a forecaster diffuses its predictions over the cell graph — and quantifies the effect on held-out cell-state forecasting under donor, batch, time and perturbation shift. Diffusion yields consistent accuracy gains, recovers a rare state that an otherwise-identical point classifier almost entirely misses, and incurs a calibration cost reported in full; a density-steered active-sampling policy converts a fixed annotation budget into rare-state coverage far above the base rate. Proved variational and spectral guarantees certify each structural property the forecaster relies upon, and every reported figure is transcribed verbatim from a single seeded artifact. The study forms one entry in a unified graph-learning program spanning single-cell biology and quantum computation.

1Background and Motivation

Single-cell measurements resolve differentiation, activation and treatment response as continuous trajectories. A recurring analysis goal is to forecast the state a cell or population will occupy under conditions absent from the training set — a donor not yet profiled, a batch run later, a later point along a developmental trajectory, an untested perturbation — precisely the regime that demands generalization off the training distribution.

Two structural features of single-cell data jointly define the problem addressed here.

Manifold concentration

Cells do not fill feature space uniformly; they concentrate near a low-dimensional manifold that single-cell pipelines routinely approximate with a k-nearest-neighbour (kNN) graph before clustering, embedding or pseudotime inference. A classifier reading features alone discards exactly the neighbourhood geometry this graph encodes.

Rarity of the decisive states

The biologically decisive states — transient progenitors, treatment-resistant clones — are typically rare. Classifiers trained to minimise average error learn to ignore them, and aggregate accuracy masks the failure. Recovering these states, and doing so with calibrated uncertainty, is the open problem.

Two long-studied levers address these features directly. Diffusing predictions over the data graph — graph-based semi-supervised learning by label propagation — lets each cell borrow statistical strength from its manifold neighbourhood. Choosing which cells to annotate — active learning — can be steered toward the sparse, uncertain regions where rare states reside, once informativeness is coupled with a manifold-density signal. Both are standard; what is missing is a clean measurement of how much the first lever actually buys under realistic shift.

This work claims no new algorithm. Its contribution is a controlled, leakage-checked measurement of the value of graph structure for single-cell forecasting, obtained by holding the available neighbourhood information fixed and toggling only whether the model diffuses over the graph. Three quantities are targeted together: held-out generalization under covariate shift, rare-state recall, and the calibration cost that recovering rare states incurs.

All experiments run on a seeded synthetic branching lineage. This is a falsifiable sanity benchmark, not a biological dataset; that scope is stated explicitly throughout and bounds every claim below.

2Mathematical Preliminaries

Fix $n$ cells embedded in a feature space $\mathbb{R}^{d}$ (a stand-in for a PCA or latent embedding of expression). The objects defined below are precisely those the forecaster and its guarantees invoke; each is stated once and reused throughout.

Cell-similarity graph, degree, Laplacian

Let $A\in\mathbb{R}^{n\times n}$ be the symmetric adjacency of the kNN cell graph: $A=A^{\top}$, $A_{ij}\ge 0$, $A_{ii}=0$, and $A_{ij}>0$ iff cells $i,j$ are joined by an undirected edge. An edge $\{i,j\}$ is present under the either-direction convention iff $j$ is among the $k$ nearest neighbours of $i$ or $i$ is among the $k$ nearest neighbours of $j$. The degree is $d_i=\sum_j A_{ij}$, collected in $D=\operatorname{diag}(d_1,\dots,d_n)$, and the combinatorial graph Laplacian is $$ L \;=\; D - A. $$ The Betti-0 number $\beta_0(A)$ is the number of connected components of the graph. In this benchmark it is monitored purely as a build sanity check.

Dirichlet energy and the graph-smoothing operator

The Laplacian quadratic form is the Dirichlet energy $$ f^{\top} L f \;=\; \tfrac12\sum_{i,j} A_{ij}\,(f_i-f_j)^2 \;\ge\; 0, \qquad L \succeq 0, $$ which penalises label disagreement across edges. Label propagation diffuses one-hot seed labels over the graph using the symmetric normalisation $S=D^{-1/2}AD^{-1/2}$ and a clamping parameter $\alpha\in(0,1)$: $$ F^{(t+1)} \;=\; \alpha\,S\,F^{(t)} + (1-\alpha)\,F_0, \qquad F^{(0)}=F_0, $$ run for a fixed number of power iterations. The implementation uses $\alpha=0.9$ and $30$ iterations; held-out cells are predicted from the row-normalised diffused scores, with the maximum class probability serving as the confidence used for calibration.

Apples-to-apples kNN baseline

The control is a plain kNN classifier in feature space, fitted on the same annotated cells with the same neighbourhood size $k$: identical neighbourhood information, without diffusion. Any measured difference is therefore attributable to the diffusion step alone, not to the mere availability of neighbours.

Metrics

Accuracy

Fraction of held-out cells classified correctly: $\mathrm{Acc}=\tfrac{1}{|\mathcal{T}|}\sum_{i\in\mathcal{T}}\mathbf{1}[\hat y_i = y_i]$.

Expected calibration error

Population-weighted gap between mean confidence and empirical accuracy over ten bins, $\mathrm{ECE}=\sum_b \tfrac{|B_b|}{n}\,\bigl|\operatorname{acc}(B_b)-\operatorname{conf}(B_b)\bigr|$. Lower is better.

Rare-state recall

Fraction of true rare cells recovered, $\dfrac{|\{i:\hat y_i=r,\;y_i=r\}|}{|\{i:y_i=r\}|}$ for the rare class $r$, set to $1$ by convention when a test set contains no rare cells.

3Problem Formulation

The task is transductive, few-label single-cell state forecasting, evaluated under covariate shift. The two defining ingredients — the scarcity of labels and the structure of the held-out set — are specified next.

Transductive few-label forecasting. Given the cell set with features $X\in\mathbb{R}^{n\times d}$ and metadata (branch/state, pseudotime, donor, batch, perturbation), only a small fraction $\rho=\texttt{label\_fraction}=0.03$ of the training pool is annotated, with a light stratification that guarantees a few labels for each present state, including the rare one. The model must predict the discrete state of every held-out cell, mirroring the workflow in which an analyst annotates a handful of cells and propagates the rest. Because all cells — labelled and unlabelled — are available at fit time, the setting is transductive.

Covariate-shift evaluation protocol. Generalization is measured under four leakage-checked split protocols, each instantiating a distinct family of distribution shift:

  • Donor — hold out whole donors (biological replicate transfer).
  • Batch — hold out whole batches (technical replicate transfer).
  • Time — train on early pseudotime, test on late pseudotime (a forward-in-time extrapolation; the hardest protocol).
  • Perturbation — hold out a whole condition (forecast an unseen perturbation).

For the three grouping splits, a programmatic check asserts that no held-out group appears in both train and test — a strict family-held-out no-leakage guarantee; the time split has no grouping variable to leak. Donor, batch and perturbation labels are assigned independently of the biological branch, so each split is non-trivial: a single donor spans multiple branches, and the model cannot succeed by memorising a donor-to-branch shortcut. Performance is reported as held-out accuracy, ECE and rare-state recall, aggregated over five master seeds with $95\%$ normal-approximation confidence intervals $1.96\,s/\sqrt{n}$.

4Principal Contributions

  1. A benchmark that isolates graph structure as the single variable. A seeded synthetic branching lineage ($n=3000$ cells, $16$ features, $8$ donors, $4$ batches, $5$ perturbations, a $\sim 4\%$ rare state) is paired with an apples-to-apples control: the graph-smoothed forecaster and the kNN baseline receive identical neighbourhood information and differ only in whether predictions diffuse over the graph. The design licenses a causal reading of every reported gain.
  2. Accuracy gains on all four shift protocols. Graph-smoothed label propagation raises held-out forecast accuracy on every split — by $0.0454$ (donor), $0.0640$ (batch), $0.0900$ (time) and $0.0385$ (perturbation), a mean gain of $0.0595$ — with the largest gain on the hardest, forward-in-time protocol.
  3. Recovery of a rare state the baseline almost never finds. The same diffusion lifts rare-state recall to $0.1520$ (donor), $0.4618$ (batch) and $0.3393$ (perturbation) from a near-zero baseline; the time split is reported as a null, and the theory explains why.
  4. A topology-aware active-sampling policy that targets the rare manifold. Scoring unlabelled cells by forecast uncertainty $\times$ inverse local density raises rare-state recall to $0.3556$ (95% CI $\pm 0.1597$) against $0.0247$ ($\pm 0.0285$) for uniform random querying — a lift of $0.3309$ — while spending $36.7\%$ of its queries on genuinely rare cells against a $4\%$ base rate.
  5. A calibration cost reported in full. ECE is consistently worse for the graph-smoothed forecaster on every split; the accuracy–calibration trade-off is stated openly rather than obscured.
  6. Self-contained theory for every property used. Each structural fact the forecaster relies upon is stated and proved: the Dirichlet-energy identity, the spectral count of components, the variational characterisation of label propagation, its exact non-identifiability limitation, and the geometric convergence and closed form of the normalised iteration the code runs.
  7. Deterministic, end-to-end reproducibility. A pip-installable package (topocell 0.1.0) regenerates every table, figure and number from a single seeded summary.json.

5Methodological Novelty

The novelty lies not in a new propagator but in the controlled isolation of where graph structure helps, paired with a density-steered sampling policy. Two design choices separate this study from prior single-cell baselines.

Graph-structure-aware propagation, controlled

Standard practice builds a kNN graph and then either reads off neighbours (the plain classifier) or clusters and embeds on it. Here the diffusion step is the only variable that differs between the two methods: both consume the same annotated cells and the same neighbourhood radius. The measured gain is therefore attributable to graph diffusion itself rather than to the availability of neighbours, and the four leakage-checked split protocols make that attribution robust across distinct families of nuisance variation.

Topology-aware active sampling

The sampling policy multiplies forecast uncertainty (the entropy of the label-propagation forecast) by inverse local density (the reciprocal of per-cell mean kNN distance, large in sparse pockets), combining the two as standardised z-scores. It thereby steers a fixed annotation budget into exactly the sparse, low-density manifold regions where rare cells reside — the regions where, by the non-identifiability result, smoothing alone cannot help. Uniform random querying, the baseline, instead spends its budget at the $4\%$ base rate.

The "topology" invoked here is deliberately shallow: kNN-graph connectivity, with the Betti-0 statistic serving as a connected-component count used only as a build sanity check. No persistent homology or higher-order invariant is employed, and the method is described as graph- and manifold-structure-aware rather than topological, to avoid overclaiming.

6Theoretical Guarantees

The statements below are transcribed verbatim from the manuscript's self-contained theory section, each annotated with a one-line reading. Together they establish that the forecaster is well posed, that its only failure mode is a structural one, and that the iteration the code runs faithfully realises a closed-form estimator.

Lemma 1 (Dirichlet-energy identity and positive semidefiniteness).

For every $f\in\mathbb{R}^{n}$, $\;f^{\top}Lf=\tfrac12\sum_{i,j}A_{ij}(f_i-f_j)^2\ge 0$, so $L\succeq 0$; more generally $\operatorname{Tr}(F^{\top}LF)=\tfrac12\sum_{i,j}A_{ij}\|F_i-F_j\|_2^2$.

Graph smoothing minimises a convex, nonnegative energy whose value is the total label disagreement across edges.
Corollary 2 (Null space of $L$ and the spectral count of components).

$L\mathbf{1}=0$, and the multiplicity of the eigenvalue $0$ of $L$ equals $\beta_0(A)$, the number of connected components; $L\succ 0$ on the orthogonal complement of its null space.

Connectivity (Betti-0) equals the dimension of the zero-energy subspace — the structural reason a component carrying no label cannot be resolved.
Proposition 3 (Monotonicity of the connectivity diagnostic).

The symmetric kNN edge set is monotone non-decreasing in $k$, $E(A^{(k)})\subseteq E(A^{(k+1)})$, so $\beta_0(A^{(k+1)})\le\beta_0(A^{(k)})$ for all $k\ge 1$.

Adding neighbours can only merge components, never split them — the property the build-time monotonicity gate asserts over $k\in\{2,4,8,16,32\}$.
Theorem 4 (Unique graph-regularised extension of labels).

For fidelity weight $\mu>0$, the objective $J(F)=\operatorname{Tr}(F^{\top}LF)+\mu\|M(F-Y)\|_F^2$ is convex with normal equations $(L+\mu M)F=\mu MY$. If every connected component contains at least one labelled cell, then $L+\mu M\succ 0$ and the minimiser $F^{\star}=\mu(L+\mu M)^{-1}MY$ is unique — the minimum-Dirichlet-energy extension of the observed labels.

Label propagation is the unique minimiser of a convex graph-regularised objective — the minimum-energy extension of the labels — whenever every component carries at least one label.
Corollary 5 (Non-identifiability of a label-free component).

If some component $C$ contains no labelled cell, then $L+\mu M$ is singular and the minimiser is not unique: it may be shifted by an arbitrary per-class constant supported on $C$ without changing $J$.

Smoothing cannot invent a label class unreachable from the annotated set — the formal mechanism behind the null rare-state recall on the time split, and the exact gap that active sampling is designed to close.
Theorem 6 (Spectrum of the normalised adjacency).

If $d_i>0$ for all $i$ and $S=D^{-1/2}AD^{-1/2}$, then $S$ is symmetric with every eigenvalue in $[-1,1]$; hence for $\alpha\in(0,1)$ the matrix $I-\alpha S$ is symmetric positive definite with eigenvalues in $[1-\alpha,1+\alpha]\subset(0,2)$.

The normalised diffusion is well posed and invertible for every clamping parameter $\alpha\in(0,1)$, with no spectral tuning required.
Theorem 7 (Geometric convergence and closed form of label propagation).

The iteration $F^{(t+1)}=\alpha S F^{(t)}+(1-\alpha)F_0$ converges to the unique fixed point $F^{\star}=(1-\alpha)(I-\alpha S)^{-1}F_0$ with $\|F^{(t)}-F^{\star}\|_F\le\alpha^{t}\|F^{(0)}-F^{\star}\|_F$, and $F^{\star}$ is the unique minimiser of the normalised energy $Q(F)=\tfrac12\operatorname{Tr}(F^{\top}(I-S)F)+\tfrac{1-\alpha}{2\alpha}\|F-F_0\|_F^2$.

The 30-step, $\alpha=0.9$ diffusion the code runs is a geometrically convergent, faithful realisation of a closed-form minimum-energy estimator, with residual error $\le\alpha^{30}$ of the initial gap.

Reading of the accuracy result

Graph diffusion helps consistently but modestly: per-split gains of roughly four to nine percentage points, largest on the hardest (time) protocol. This is the magnitude a single diffusion step should produce — a real, repeatable improvement, not a categorical change in capability, and the page does not present it as one.

Reading of the rare-state result

The larger effect is rare-state recovery. The time-split null is reported as-is, and its cause is structural: forward extrapolation in pseudotime places the rare late-tip cells entirely beyond the training distribution, leaving them in a label-free region that diffusion provably cannot reach. Wide intervals on per-split rare recall follow from small absolute rare counts and caution against over-reading any single point estimate.

9Significance and Impact

Within a controlled benchmark, conditioning single-cell forecasts on cell-graph connectivity demonstrably helps — consistently across four leakage-checked shift protocols, and most decisively by recovering a rare state that an otherwise-identical point classifier almost entirely misses.

For computational biology, the result is a clean, falsifiable statement of where graph structure pays off under distribution shift, together with a density-steered annotation policy that converts a fixed labelling budget into rare-state coverage far above the base rate — directly relevant to the practical problem of allocating scarce annotation effort. For machine learning under covariate shift more broadly, the controlled isolation — identical neighbourhood information, diffusion as the single toggled variable — supplies a transferable template for attributing a generalization gain to one mechanism rather than to a tangle of confounded design choices. The accompanying theory certifies that the deployed iteration faithfully realises a closed-form minimum-energy estimator, so the observed behaviour is grounded in the method rather than incidental to it.

The decisive next step is to repeat exactly this comparison on real scRNA-seq and perturbation-screen data, for which the repository ships intentionally unexercised ingest stubs. The same protocol then extends naturally to a learned message-passing propagator in place of fixed-point label propagation, and to calibration-aware training that targets the ECE penalty directly.

Limitations (stated in full)

(1) Synthetic data. All numbers come from a generated branching lineage, not biological measurements; no biological discovery is claimed, and the magnitudes need not transfer to real scRNA-seq.
(2) Modest accuracy gains. The accuracy improvement is $0.04$–$0.09$ per split (mean $0.0595$); the strongest effect is rare-state recall, and even that is null on the time split.
(3) Calibration cost. Graph smoothing worsens ECE on all four splits, so its probabilities are less trustworthy as confidences without recalibration.
(4) Shallow "topology." The only topological object is kNN-graph connectivity; the Betti-0 statistic is a connected-component count, constant at one at this scale. No persistent homology is used.
(5) Wide intervals on rare metrics. Few rare cells per held-out group produce wide per-split rare-recall confidence intervals (up to $\pm 0.3358$).
(6) Single benchmark family. One synthetic generator cannot stand in for the diversity of real datasets; external validation remains the decisive test.

10Reproducibility

The reference implementation is the pip-installable package topocell 0.1.0 at submission/code (CPU-only: NumPy, SciPy, scikit-learn, NetworkX, Matplotlib). A single seeded results/summary.json is the sole source of truth for every table, figure and numerical claim on this page; the manuscript's Table 1 is transcribed verbatim from the generated main_results.tex. No value here is hand-entered.

# 1. install the CPU reference package (editable)
conda env create -f environment.yml && conda activate topocell
pip install -e .            # installs the `topocell` package
# pip install -e .[gpu]    optional torch GPU/GNN backend

# 2. reproduce every number, table and figure
make test        # synthetic structure - no-leakage splits - Betti-0 monotonicity - metric bounds
make full-run    # reported-scale config -> results/summary.json (~20.8 s, 320.6 MB peak)
make tables      # results/main_results.tex
make figures     # figures/fig_lineage.pdf, figures/fig_forecast.pdf

# or one command:
bash scripts/reproduce_all.sh full   # reported scale

The full-scale run is launched by scripts/run.py on configs/full.yaml, writing to results/. Three integrity gates must pass before any metric is recorded: grouping splits are leakage-free, the rare state is present, and the Betti-0 component count is monotone non-increasing in $k$. A failed gate halts the run rather than emitting a number.
